# Claude Code Instructions for NeuralGraphDB

## Git Commit Preferences

- **Do NOT add Co-Authored-By lines** to commit messages
- Follow conventional commit format: `feat:`, `fix:`, `docs:`, `chore:`, etc.
- Keep commit messages concise but descriptive

## Project Context

- This is a Rust graph database project
- Sprint-based development (currently in Phase 7)
- Use `cargo test` to verify changes
- Documentation lives in `docs/` folder
